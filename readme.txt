Simple one-click restore post when writing

== Description ==

Undo Box adds a simple metabox to your post editor, allowing you to quickly restore the last (auto)saved version of a post.

Reverting again restores the initial version.

= Features =

* One-click revert
* Quick link to compare current and latest revision
* Quick link to all revisions meta box (and shows it when hidden)

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
